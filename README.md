# com.dave6.character-kit
**Unity** 에서 캐릭터 컨트롤러를 구축하기 위한 가벼운 패키지

## Requirements

- Unity Input System
- Cinemachine 3.1+
- Timer Package
- Unity Util Package
- State Machine Package

## Features

기본 캐릭터 컨트롤러 구성 요소
간단한 3인칭 카메라 타겟 설정
상태 기반 캐릭터 로직
샘플 프리팹 포함


## Quick Start

1. **Samples** 에 제공된 캐릭터 프리팹을 씬에 추가
2. 씬에 있는 `Main Camera` 에 `Cinemachine Brain` 컴포넌트 추가 및 `MainCamera` 태그 설정
3. 필요에 따라 `Profile` 구성
